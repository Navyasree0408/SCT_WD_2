# Stopwatch Web Application - SkillCraft Technology
This repository contains the source code for the **Stopwatch Web Application** task, completed as part of the Web Development Internship at [SkillCraft Technology](link-to-skillcraft-website-or-linkedin). 
## 📝 Project Overview
The objective of this project was to create an interactive and user-friendly stopwatch web application. The application allows users to accurately measure and record time intervals, demonstrating core JavaScript timing events and dynamic DOM manipulation.
## ✨ Key Features
* **Start & Pause:** Initiate and halt the stopwatch with precision.
* **Reset Functionality:** Instantly clear the timer and all recorded data.
* **Lap Tracking:** Record and display specific time intervals (laps) dynamically without stopping the main timer.
* **User-Friendly Interface:** Clean, modern, and responsive design for an optimal user experience across devices.
## 🛠️ Tech Stack
* **HTML5:** Structures the stopwatch interface and lap list.
* **CSS3:** Styles the application, utilizing hover effects and responsive layout techniques.
* **JavaScript (ES6):** Manages the core logic, including the `setInterval` methods, time calculations (hours, minutes, seconds, milliseconds), and dynamic HTML generation for laps.
